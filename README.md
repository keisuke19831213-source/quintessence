# QUINTESSENCE — 内接するプラトン立体

一つの正六面体(立方体)の中に、残る四つのプラトン立体がすべて内接する
3次元メタトロン・キューブのインタラクティブ WebGL2 アート。

視線を立方体の**体対角線**に重ねると、頂点の正射影がちょうど 13 点
(フルーツ・オブ・ライフ)となり、二次元のメタトロン・キューブが結晶する。

## 幾何学

立方体の半辺長を 1、φ を黄金比とすると:

| 立体 | 内接のしかた |
|---|---|
| 正四面体 ×2 | 立方体の頂点 8 つを交互に取る(星型八面体/マカバ) |
| 正八面体 | 立方体の 6 つの面心が頂点 |
| 正二十面体 | (0, ±1/φ, ±1) の巡回置換 — 12 頂点が面上に載る |
| 正十二面体 | 標準座標を 1/φ 倍 — 12 頂点が面上に載る |

体対角線 (1,1,1) 方向の正射影では、立方体の頂点が外側六角形(半径 2r)、
八面体の頂点が内側六角形(半径 r)、中心と合わせて 13 円配置になる
(r = √(2/3))。

## 技術

- 純 WebGL2・単一 HTML ファイル・依存ゼロ
- HDR パイプライン:RGBA16F → ブライトパス → 2 スケールのガウシアンブルーム → ACES トーンマップ
- ネオンライン:スクリーン空間 SDF(コア+ハロー)による加算発光
- ドリーズーム:対角線への整列度に応じて FOV 42°→3.1°(正射影に漸近)
- 整列時のみ現れる 13 円 + 78 本の結線オーバーレイ
- WebAudio 生成音響(パッド+五音音階のプラック+整列ベル)

## 操作

- ドラッグ:回転 / ホイール・ピンチ:遠近
- `1`–`5`:各立体の表示切替
- `M`:体対角線ビューへ整列(近傍では緩やかに吸引)
- `A` 自動回転 / `P` 配色(AURORA・SOLAR・VOID)/ `S` 音 / `H` 解説
- クリック:波動を放つ

## 実行

`index.html` をブラウザで開くだけ。
